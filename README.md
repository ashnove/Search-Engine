# Search-Engine

A responsive search engine backend designed with Trie and Heap modelled over predeï¬ned data sets.

To run the application, used command:
```
./engine -d <filename.txt> -k <number of doc>
```

Features
-----

General Search:
```
/search <text>
```

Term Find:
```
/tf <document number> <text>
```

Document Find:
```
/df <text>
```
